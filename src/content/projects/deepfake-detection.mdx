# DeepTruth — Spatial-Temporal Deepfake Video Detection

A high-accuracy machine learning and deep learning (ML/DL) classification system. Designed to identify sophisticated frame-manipulated media using a two-stage sequential deep neural network.

**Key Deep Learning Paradigm:** A single image deepfake model fails to capture inconsistencies across frame swaps. By utilizing a **ResNet CNN spatial extractor** in parallel with an **LSTM temporal sequence tracker**, the pipeline detects minute flickering anomalies that occur during artificial face transitions.

---

## Engineering Design Review & Machine Learning Decisions

### 1. Spatial Extractor Selection (Transfer Learning)
* **Decision:** We deployed a pre-trained **ResNet-50** Convolutional Neural Network as our spatial feature extractor instead of training a CNN from scratch.
* **Rationale:** Spatial CNNs require massive volumes of general image parameters to capture basic details (edges, lighting, textures). Leveraging a ResNet model pre-trained on ImageNet significantly decreases local training times and captures structural face attributes instantly.
* **Trade-off:** ResNet-50 generates a dense feature dimension, increasing downstream network ingestion overhead, but guarantees stable, high-fidelity spatial representation.

### 2. Recurrent Temporal Tracking (LSTM Layer)
* **Decision:** Leveraged a **Long Short-Term Memory (LSTM)** network to process the sequential feature vectors output by the CNN.
* **Rationale:** Deepfakes frequently display anomalies in temporal movement (e.g., eye blinking intervals, mouth movement consistency across frames). LSTMs use gated memory cells to track temporal dependencies across long sequences, making them highly effective at catching artificial splice anomalies.
* **Trade-off:** Recurrent units cannot execute parallel GPU training as effectively as newer Transformer architectures, but they maintain a highly efficient, lightweight computational footprint for batch inferences.
